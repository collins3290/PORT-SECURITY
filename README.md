# PORT-SECURITY AND LAN HARDENEN
<body>
  <h2>How to configure port security Violation {shutdown} in packet tracer</h2> <br>
1. Open the Switch CLI

Click the switch
Go to the CLI tab
Press Enter
Enter privileged mode <BR>
switchport mode access <BR>
switchport port-security <BR>
switchport port-security maximum 1 <BR>
switchport port-security violation shutdown <BR>
